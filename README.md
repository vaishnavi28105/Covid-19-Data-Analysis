# COVID-19 Data Analysis

## 📌 Overview
This project analyzes COVID-19 data to understand trends in cases, recoveries, and deaths.

## 📊 Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization using graphs and charts

## 🛠️ Tech Stack
- Python
- Pandas
- Matplotlib
- Seaborn

## 📂 Dataset
COVID-19 dataset sourced from Kaggle.

## 📈 Key Insights
- Identified trends in COVID-19 spread and recovery rates
- Visualized patterns using graphs for better understanding

## 🚀 How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the notebook/script
